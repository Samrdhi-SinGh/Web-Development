HTML Code & Pre tag

HTML Code is a Semantic tag used for displaying code snippets. It's used for showcasing Codes and it preserves the break lines and spaces.

HTML Pre tag is used to show a line as it is written with the same space gap and line breaks. It's used to print the same format a line is written in.
