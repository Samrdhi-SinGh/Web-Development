Three ways in CSS :

Inline CSS :
Inline CSS is a way of using CSS inside HTML file using 'style attribute' inside 'body tag'.
Internal CSS :
Internal CSS is a way of using CSS inside HTML file using 'style tag' in 'head tag'.
External CSS :
External CSS is a way of using CSS by making another file of CSS and connect it with HTML file with 'link tag' in 'head tag'.
