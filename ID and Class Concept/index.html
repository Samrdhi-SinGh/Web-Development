ID and Class

ID (#) and Class (.) both are Attributes.

ID: An ID is an attribute an unique Identifier assigned only to one HTML Element. It is often used for unique styling and JavaScript Manipulation.

Class: A Class is an attribute which can be assigned to Multiple HTML Element for styling them all at once. Classes are not unique. And also an Element can have multiple Classes too.
